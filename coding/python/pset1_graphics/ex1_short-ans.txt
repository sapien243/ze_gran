
(1.a)
    An object attribute is a variable that is associated with a class.
    The variable attribute1 in exercise1.py is an attribute of obj.

(1.b)
    A local variable is a variable associated with a function or part of a program.
    The variable test is a local variable of the main function in exercise1.py.


(2)
    A constructor is a method of an object that is called when the object is created.
    The method __init__(self, initial) is a constructor for the object obj in exercise1.py.


(3)
    Calling a method follows the format 'class_variable.class_method()'.
    Eg. calling obj's attribute do_something() in line 32 of exercise1.py
